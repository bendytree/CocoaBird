  2. Installation & Activation

    1. Step 1

      In the WordPress admin area, navigate to 'Appearance » Themes'.

    2. Step 2

      Click the tab marked 'Install Themes'. Then click 'Upload' on the submenu.

    3. Step 3

      Click 'Choose File' under the heading 'Install a theme in .zip format'. In the window that opens, select the zip file named 'Callisto_Theme' in your Themeforest download.

      Click 'Install Now'. The installation may take a few minutes or more.

    4. Alternate Option (FTP)

      If you have an FTP client installed and know the FTP login details for your site, you may prefer to upload the theme via FTP.

      If so, unzip the file named 'Callisto_theme_1.0.zip' (the version number may no longer be 1.0 due to updates) and connect via FTP to your site's WordPress install (usually the site root, unless you have opted to install WordPress in a directory).

      Finally, upload the unzipped 'Callisto_theme_1.0' folder to 'wp-content » themes'. The Callisto theme will now be available to select from your WordPress admin area.

  2. Setting Up the Homepage

    1. Step 1

      Create a new page by navigating to 'Pages » Add New.' Give this a title of 'Home' and assign it the template 'Home' in the box on the right titled 'Page Attributes.' Leave the content of this page blank for now, and click 'Publish.'

    2. Step 2

      In the WordPress admin area, navigate to 'Settings » Reading'. For the setting named 'Front Page Displays' select 'A Static Page'.

      For the dropdown titled 'Front Page,' select the Home page you just created and click 'Save.' Leave the dropdown titled 'Posts Page' blank - you will set this in the next section: 'Setting Up the Blog.'

    3. Step 3

      In the Callisto Options Panel, open the 'Home Page' section. Here you can set your welcome tagline and set the content & layout of the homepage.

      For the dropdown titled 'Homepage Content,' decide whether you would like blog posts or custom content on your homepage. If you select 'Both,' your custom content will appear first and the blog posts below.

    4. Custom Content

      In Step 1 of this section, when creating the homepage it was published with the content left blank. If you set the 'Homepage Content' dropdown in the Options Panel (see previous step) to 'Custom Content' or 'Both,' any content that you now add to that blank page will be added to your homepage, below the featured area.

      To enable maximum customisability, the homepage is a completely blank template - please read our section on 'Working with Page Layouts' to see how to effectively add custom content to the homepage.

      Please also view our screencasts as these demonstrate how to set up custom content for the homepage in greater detail.

  3. Setting Up the Featured Area

    1. Step 1

      The featured area is the large area on the homepage for displaying your primary content. By default it is a slider, but it can also be an accordion or a large image that links to a single portfolio item or blog post.

      To set up the featured area, you must first create a category for featured content to be pulled from. Navigate to 'Posts » Categories' and choose a category name for this featured content (this name can be anything). Next, click 'Add New Category.'

    2. Step 2

      In the Callisto Options Panel, open the 'Featured Area' section. For the dropdown titled 'Select a Featured Category', select the category that you created in the previous step, and click 'Save All Changes.'

      Important: You must enter a height for the featured area at this point (default is 380px). The width of the featured area is fixed to 970px, so please ensure that all featured images for posts/items in the featured category are at least 970px by 380px (or whatever you set the height to). If they are larger than this that is OK - they will be automatically resized when they are uploaded.

      Please also note that if you uploaded featured items and then changed the height of the featured area afterwards, these images will be the wrong size. To fix this, you can either re-upload the images or use the 'Regenerate Thumbnails' plugin included in the Themeforest download.

    3. Step 3

      The featured area is now set up! Let's add some content to check everything is working as expected.

      Create a new post or portfolio item that you would like to be featured on the front page. While creating this post/portfolio item, in the box on the right titled 'Categories' select the category that you create in the previous step.

    4. Step 4

      Before publishing this post/portfolio item, you must add a featured image (this is what will be displayed on the slider/accordion). To do this, click 'Set Featured Image' in the box on the right titled 'Featured Image.'

      This will open a modal window through which you can upload your picture. Click 'Select File' and choose the image from your local file system.

    5. Step 5

      For the Featured Area, you can leave all the image fields blank - just click 'Use as Featured Image,' then click 'Save all Changes' and close the modal window with the cross in the upper-right.

    6. Step 6

      Publish the post/portfolio item and visit your homepage to verfiy that the featured area is working correctly. That's it!

      Please note that you must have completed the earlier section 'Setting Up Your Homepage' in order for the homepage to display the featured area.

      Please also note that if your featured area is set to 'Accordion' in the Callisto Options Panel, there must be at least five posts in the featured category for it to pull from - otherwise, there will be blank spaces.

  4. Setting Up the Blog

    1. Step 1

      Create a new page by navigating to 'Pages » Add New.' Give this a title of 'Blog' (or anything you would like your blog to be called). Leave the content of this page blank and click 'Publish.'

    2. Step 2

      In the WordPress admin area, navigate to 'Settings » Reading'. For the dropdown titled 'Front Page,' select the Blog page you just created and click 'Save.'

    3. Step 3

      In the Callisto Options Panel, open the 'Blog Settings' section. Here you can set the layout of the blog page, with granular control over columns/number of posts/sidebar, etc.

      The last option, titled 'Display Sidebar on Blog Entry?', is the only one that does not refer to the blog overview. Instead it refers to whether a sidebar will be present when viewing single posts.

      Once you have five or six posts published, it should be much easier to find out which type of layout works best with your content. Creating a few test posts with dummy content/featured images could help you assess how the blog options affect the blog layout.

    4. Adding Blog Posts

      Create new blog posts by navigating to 'Posts » Add New.'

      Make sure to add featured images to these items - the process is identical to that covered in steps 4-5 in the 'Setting Up the Featured Area.'

  5. Setting Up the Portfolio

    1. Step 1

      Create a new page by navigating to 'Pages » Add New.' Give this a title of 'Portfolio' (or anything else you would like it to be called, e.g. 'Projects' or 'Work') and assign it the template 'Portfolio' in the box on the right titled 'Page Attributes.' Leave the content of this page blank and click 'Publish.'

    2. Step 2

      In the Callisto Options Panel, open the 'Portfolio Settings' section. In the input for 'Your Portfolio Name,' either leave this as 'Portfolio' or enter some other description, e.g. 'Projects' or 'Work.' This will be used for the URL slug for all portfolio pieces. It will also take the place of the text 'Portfolio' in various places in the WordPress admin area.

      You can use punctuation/spaces/multiple words for this setting if you like - it will be cleaned up and hyphenated using regular expressions to create a URL-friendly slug.

    3. Step 3

      If you changed the 'Portfolio' text in Step 2 to something that better matched your content, you must go to 'Settings » Permalinks' in the WordPress admin area. Just entering this section should flush the old Portfolio permalinks so that your custom ones will work. (You can just close the page or navigate back to the WordPress Dashboard after visiting 'Settings » Permalinks').

      Important: If you ever change your 'Portfolio' name again after first setting it up, you must repeat this step in order to get the Permalinks to resolve properly. (Otherwise you may hit 404s.)

    4. Step 3

      In the Callisto Options Panel, open the 'Portfolio Settings' section. Here you can set the layout of the portfolio page, with granular control over columns/number of posts/sidebar, etc.

      The last option, titled 'Display Sidebar on Single Portfolio Entry?', is the only one that does not refer to the portfolio overview. Instead it refers to whether a sidebar will be present when viewing single portfolio items.

      Once you have five or six portfolio items published, it should be much easier to find out which type of layout works best with your content. Creating a few test portfolio items with dummy content/featured images could help you assess how the portfolio options affect the layout.

    5. Adding Portfolio Items

      Create new portfolio items by navigating to 'Portfolio » Add New.'

      Make sure to add featured images to these items - the process is identical to that covered in steps 4-5 in the 'Setting Up the Featured Area.'

  7. Configure Your Widgets

    1. Overview

      This theme has a widgetised footer and sidebar. The footer is designed to hold four widgets (see theme demo for widget layout); there is no real limit to the amount of widgets the sidebar can hold.

      This theme comes with four custom widget: Callisto Sidebar Adverts, Callisto Sidebar Google Map, Callisto Twitter Feed & Callisto Flickr Stream.

      Widgets are activated in the usual manner - drag them to the sidebar/footer, fill in the details and click 'Save.'

    2. Callisto Flickr Stream

      When activating the Flickr widget, choose a title and check idGettr to fetch the Flickr ID. When selecting the number of images, multiples of three fit best in the sidebar and multiples of two fit best in the footer.

    3. Callisto Sidebar Google Maps Widget

      When activating the Google Maps widget, choose a title and a pixel height (250 will make the map square).

      To find the right Google Maps Link, open the Google Maps website in a separate tab and find the relevant map area. Click 'Link' in the upper-right of the window and copy the first link (titled 'Paste link in email or IM'). Paste this link into the widget input.

    4. Callisto Twitter Feed

      When activating the Twitter widget, choose a title, enter your username and select the number of tweets to be displayed.

    5. Callisto Sidebar Adverts

      When activating the sidebar adverts widget, first choose a title.

      Next, go to the Callisto Options Panel and open the 'Sidebar 125px Ads' section. For each advert that you would like to show in your sidebar (up to a possible four), upload a 125px by 125px advert image and enter the link to which it should direct. You can also select whether these advert links should open in a new tab.

      The theme decides which adverts to display based upon whether anything has been entered in their 'Link' input in the Callisto Options Panel. So if you only upload one or two adverts, only these will be displayed - there will not be blank spaces in the layout to fill up the four.

      Included in the Themeforest download is a sample 125px by 125px image that displays the text 'Advertise Here,' styled to suit the theme. If looking for site advertising, you could upload this image in the advert slots and link to a page on your site (which you would first create) that could list your contact details, fees, audience demographic, hits per month, etc.
